# wangEditor for React

[wangEditor](https://www.wangeditor.com/v5/) React 组件。

## 使用

参考[文档](https://www.wangeditor.com/v5/guide/for-frame.html#react)。

## 开发

先打包 packages/editor
  - `cd ../editor`
  - `yarn dev` 或 `yarn build`

再操作当前包
  - 打包 `yarn dev` 或 `yarn build`
  - 或运行 example `yarn example`
